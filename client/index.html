About OSRS LLM

Built specifically for the OSRS community with comprehensive game knowledge from various sources.

Designed to run on basic consumer hardware including lower VRAM GPU's, OSRS LLM delivers expert answers about quests, items, skills, and game mechanics without relying on cloud services.

Built on Phi-3.5-mini-instruct

Microsoft's Phi-3.5-mini is a lightweight open model built upon high-quality datasets including synthetic data and filtered publicly available websites. It excels at reasoning and instruction adherence with robust safety measures.

  • 128K token context length for long conversations
  • Runs on consumer hardware (12GB VRAM)
  • Enhanced with supervised fine-tuning and DPO
  • Customizable training pipeline for more context
